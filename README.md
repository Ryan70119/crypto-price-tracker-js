# Crypto Price Tracker (JavaScript)

A simple JavaScript tool that fetches cryptocurrency prices from the CoinGecko API.

## Features

* Fetch Bitcoin price
* Fetch Ethereum price
* Uses public crypto API
* Lightweight JavaScript script

## Usage

Run the script:

node priceTracker.js

Example output:

Bitcoin price: 67000 USD
Ethereum price: 3500 USD

## API Source

Prices are retrieved from the CoinGecko public API.
